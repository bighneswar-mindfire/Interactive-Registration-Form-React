# Interactive Registration Form

## Description

This project demonstrates a simple and structured form validation system built using React and Typescript. It captures user input, validates fields, and ensures proper data handling with a clean UI.

---

## Features

- Input form with validation
- Type-safe data handling using TypeScript
- Real-time error handling
- Handles unchecked inputs safely (e.g., gender selection)
- Clean and responsive UI

---

## Tech Stack

- React
- CSS
- TypeScript
